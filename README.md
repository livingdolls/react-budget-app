## Run Locally

Run backend https://github.com/livingdolls/budget-app.git

Clone the project

```bash
  git clone https://github.com/livingdolls/react-budget-app.git
```

Go to the project directory

```bash
  cd react-budget-app
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
